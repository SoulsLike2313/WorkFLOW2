# TikTok Automation Base

Базовый desktop-клиент (PySide6 + Playwright), который:
- подключается к уже открытому профилю антидетект-браузера по CDP;
- запускает сценарии: просмотр ленты, сбор метрик, комментарии, переходы по профилям, загрузка видео;
- сохраняет события/статистику в JSONL.

## Важно
- Используйте только для своих аккаунтов и в рамках правил TikTok/антидетект-провайдера.
- При первом запуске тестируйте с маленькими лимитами (2-3 действия).
- Селекторы TikTok могут меняться, это база для доработки.

## Установка
```powershell
cd E:\CVVCODEX\tiktok_automation_app
python -m venv .venv
.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m playwright install chromium
```

## Запуск
```powershell
cd E:\CVVCODEX\tiktok_automation_app
.venv\Scripts\Activate.ps1
python app.py
```

Без дополнительного консольного окна:
- двойной клик по `run.bat` (запуск через `pythonw.exe`);
- полностью тихий старт: двойной клик по `run_silent.vbs`.

## Как использовать
1. Запусти профиль в антидетект-браузере так, чтобы был доступен CDP endpoint (`http://127.0.0.1:PORT` или `ws://...`).
2. Перетащи JSON профиля/endpoint в drop-зону или вставь CDP URL вручную.
3. Укажи URL TikTok-профиля, включи нужные сценарии и лимиты.
4. Нажми `Start`.
5. Логи и результаты будут в папке `output`:
   - `session_events.jsonl`
   - `session_stats.jsonl`

## Что доработать следующим шагом
- Конкретный адаптер под ваш антидетект (AdsPower, Dolphin, Indigo и т.д.) для автоматического старта профиля.
- Более надежные селекторы и fallback под разные языки интерфейса TikTok.
- Планировщик задач, прокси-профили и антидетект-проверки перед стартом.
