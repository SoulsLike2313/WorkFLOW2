# Media Mode Foundation

## Scope
Architecture is prepared for multimodal localization workloads:
- text
- subtitles
- audio
- video (metadata-level in current stage)
- image text candidates (heuristic level)
- containers/archives

## Working
- Asset manifest with media typing and relevance scoring.
- Content units with language/content/speaker/scene metadata.
- Audio analysis summaries and transcript segment linkage.

## Partial / Foundation
- Video semantic extraction: foundation only.
- OCR for image text: candidate detection only.
- Universal format support: not implemented.
