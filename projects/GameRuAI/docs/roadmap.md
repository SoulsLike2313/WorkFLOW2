# Roadmap (Core Platform Buildout)

## Current Stage (Implemented in this step)
- Multimodal-ready core scaffolding connected to working MVP.
- Translation package architecture with evidence routing.
- Source-of-truth registry and learning evidence store.
- Audio analysis + sync planning foundation persisted in DB.
- New product review labs in UI.

## Next Stage
- Expand content understanding for subtitle/audio/video relation quality.
- Add richer external reference providers (still evidence-only, not hidden truth).
- Improve sync risk scoring with better subtitle readability and timing heuristics.
- Add safer migration helpers for legacy DBs.

## Later Stage
- Optional heavier local models for translation and speech analysis.
- Image OCR module and video subtitle extraction plugins.
- Advanced rebuild planners for media-specific export targets.

## Non-Claims
- No production universal localization claim at this stage.
- No production speech cloning claim.
- No full real-time streaming pipeline claim.
