# UI Direction Confirmation

## Confirmed Direction
- Compact operations UI over decorative density.
- Stable primary actions at all supported scales/sizes.
- High transparency of pipeline status and diagnostics.
- Deterministic QA loop via machine-readable artifacts.

## Non-Goals for This Layer
- No visual redesign mandate.
- No framework migration.
- No fake screenshot polishing without measurable fixes.

## Acceptance Marker
A direction remains valid only if `ui_validate` is reproducible and reports trend toward fewer critical/major findings.
