# Project Template

Use this template only through the workspace generator:
- `python scripts/new_project.py`

Generator applies one of preset structures:
- `desktop_app`
- `backend_service`
- `tool`
- `prototype`
- `library`

Every generated project gets:
- unique runtime namespace
- unique port range
- project-scoped runtime/state paths

Do not create ad-hoc projects outside generator + validator workflow.
