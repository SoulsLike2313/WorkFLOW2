# EXECUTION NODE CAPSULE ADDENDUM V1

Статус документа: addendum к
`docs/review_artifacts/ULTIMATE_TRANSFER_CAPSULE_SYSTEM_V1/`.

Не является:
- новой sovereign capsule системой;
- заменой базового transfer capsule;
- переносом Palace/Throne/Emperor authority.

Роль узла:
- ASTARTES-class execution node (вкусовая роль, не authority-роль);
- сильный исполнитель;
- не интерпретатор верхнего канона;
- не holder суверенного proof.

Обязательная граница:
- main machine сохраняет верхний canonical truth и Emperor proof;
- execution node выполняет bounded work, затем возвращает результат bundle-путем.

Эскалация обязательна, если требуется:
- foundation/canon rewrite;
- authority reinterpretation;
- palace/throne transfer;
- изменение верхних governance законов.
