# EXECUTION NODE SERVER MISSION SCOPE V1

Текущая миссия ноутбука: server-first exploration.

В scope:
1. поиск доступного server контура;
2. попытки корректного подъема;
3. проверка доступности и базовой работоспособности;
4. фиксация ограничений, рисков и blocker-ов;
5. подготовка evidence для main machine.

Вне scope:
1. объявление server-архитектуры "готовой", если это не доказано;
2. изменение верхней palace/authority модели;
3. неподтвержденные claims о multi-node readiness.

Форма результата:
- что пытались;
- что реально поднялось;
- что не поднялось;
- почему;
- какие owner decisions нужны дальше.
