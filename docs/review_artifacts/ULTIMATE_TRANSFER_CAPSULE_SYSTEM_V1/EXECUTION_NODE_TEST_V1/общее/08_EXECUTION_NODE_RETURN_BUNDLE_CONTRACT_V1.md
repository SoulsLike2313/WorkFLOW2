# EXECUTION NODE RETURN BUNDLE CONTRACT V1

После миссии execution node обязан вернуть bundle на main machine.

Минимальный контракт:
1. canonical review root для шага;
2. `chatgpt_transfer/` пакет (core/visual/optional);
3. явный validation report;
4. truth/gaps report;
5. changed surfaces list;
6. API/runtime smoke evidence;
7. визуальные evidence при UI-изменениях.

Обязательный отчет в bundle:
- what worked;
- what failed;
- what remains unknown;
- what owner must decide.

Формат передачи:
- manual handoff bundle from laptop to main machine;
- без claims, что authority переносится вместе с bundle.

Принцип:
- bundle = evidence + handoff,
- не replacement sovereign truth.
