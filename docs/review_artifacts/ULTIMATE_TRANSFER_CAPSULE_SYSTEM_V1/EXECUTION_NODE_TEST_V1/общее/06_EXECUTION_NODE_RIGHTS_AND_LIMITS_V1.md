# EXECUTION NODE RIGHTS AND LIMITS V1

## Rights (allowed)

1. bounded execution in local repo copy;
2. server exploration / bring-up attempts;
3. diagnostics, smoke checks, validation;
4. bounded code changes in approved scope;
5. review artifacts + transfer package generation;
6. return bundle handoff to main machine.

## Limits (forbidden)

1. Emperor authority reassignment;
2. Golden Throne authority relocation;
3. sovereign canon rewrite;
4. foundation governance overwrite;
5. silent assumption on authority-sensitive choices.

## Escalation triggers

Эскалация обязательна, если требуется:
- foundation-layer change;
- authority model change;
- cross-machine sovereignty decision;
- operation outside current bounded mission.

## Canon reminder

Emperor не "становится" и не "переносится" как mutable статус.
На execution node может провалиться только proof/доступность, но не меняется source authority.
