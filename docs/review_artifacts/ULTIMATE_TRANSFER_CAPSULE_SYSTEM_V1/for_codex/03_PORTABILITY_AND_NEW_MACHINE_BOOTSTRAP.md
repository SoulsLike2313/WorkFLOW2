# 03_PORTABILITY_AND_NEW_MACHINE_BOOTSTRAP

Portable bootstrap order for new/clean/local machine:

1. restore/open canonical repo
2. open capsule root:
- docs/review_artifacts/ULTIMATE_TRANSFER_CAPSULE_SYSTEM_V1/
3. read:
- 00_CAPSULE_ENTRYPOINT.md
- 01_FOUNDATION_IMMUTABLE_CANON.md
- 02_MUTABLE_ACTIVE_STATE.md
4. verify current point from:
- MUTABLE_TRACKER.json
- 08_CURRENT_POINT_VERIFICATION.md
- 06_CURRENT_WORK_LADDER_AND_RECENT_CHAIN.md
5. reconstruct bounds from:
- 06_CODEX_SEED_EXECUTION_BOUNDARY.md
- 07_CODEX_RECONSTRUCTION_CHECKLIST.md
6. only then execute next bounded delta

Boundary:
- new machine transport is not equal to sovereignty transfer.
