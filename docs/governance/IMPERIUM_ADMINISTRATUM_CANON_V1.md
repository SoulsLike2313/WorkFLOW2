# IMPERIUM_ADMINISTRATUM_CANON_V1

Status:
- class: `foundation_contract_organ`
- mutability: `immutable_by_default`
- change_authority: `EMPEROR_ONLY`
- implementation_state: `active_contract_surface`

Purpose:
- Administratum converts owner intent into strict executable contracts.
- Administratum prevents silent machine guesswork.
- Administratum enforces stop-and-ask discipline when ambiguity is high.

Contract minimum:
1. `intent`
2. `scope`
3. `must_preserve`
4. `unknowns`
5. `assumptions`
6. `stop_triggers`
7. `acceptance_criteria`
8. `risk_cost_estimate`
9. `owner_escalation_rule`

Execution law:
1. large bounded execution without contract is forbidden,
2. unknowns/assumptions above threshold must block execution,
3. block means ask owner, not invent a path,
4. contract status must be dashboard-visible.

Boundary:
1. Administratum does not define sovereign authority rank,
2. Administratum does not replace Custodes,
3. Administratum does not perform anti-heresy remediation itself.

Truth rules:
1. contract completeness must be explicit,
2. missing fields must be explicit,
3. assumption pressure must be explicit,
4. no fake "contract complete" when critical fields are absent.
