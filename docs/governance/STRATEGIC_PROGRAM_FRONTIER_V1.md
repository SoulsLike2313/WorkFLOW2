# STRATEGIC_PROGRAM_FRONTIER_V1

Статус:
- version: `v1`
- memory_role: `frontier_boundary_contract`
- labels_used: `PROVEN | ACTIVE | MAPPED | FROZEN | NOT YET IMPLEMENTED`

## 1) Что такое active program

`ACTIVE` программа:
1. имеет текущую owner-approved цель;
2. имеет рабочий execution контур;
3. имеет обязательные gates и evidence outputs.

Для текущего этапа active:
1. TikTok Agent owner-steered hardening program;
2. Local-Only Factory Observation Layer V1 bootstrap.

## 2) Что такое next unlocks

`MAPPED` next unlock:
1. формально описан;
2. еще не разрешен к исполнению;
3. имеет условия открытия.

## 3) Что такое frozen future

`FROZEN`:
1. стратегически важные направления дальнего горизонта;
2. хранятся в памяти системы;
3. не допускаются к активации до закрытия unlock conditions.

## 4) Что такое forbidden drift

Forbidden drift:
1. прыжок в дальние программы, минуя текущий frontier;
2. раздувание scope без gate approval;
3. подмена доказательной работы “стратегическими рассказами”.

## 5) Как не потерять дальнюю цель и не расплыться

1. north star фиксируется отдельно и не меняется на каждом таске;
2. roadmap отделяет `ACTIVE/MAPPED/FROZEN`;
3. каждое открытие новой стадии проходит через strategic activation gate.

## 6) Frontier lock rule

Пока active frontier не закрыт по его gates:
1. frozen программы не активируются;
2. mapped программы не считаются текущим execution scope.
