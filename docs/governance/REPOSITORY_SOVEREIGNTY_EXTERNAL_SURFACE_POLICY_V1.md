## Core rule

1. local canonical repo is sovereign source,
2. safe mirror and external Git surfaces are optional non-sovereign derivatives,
3. default canon path must not depend on safe mirror.

## External surface rule

1. safe/export creation is manual,
2. safe/export creation is sovereign decision driven,
3. external surface cannot override or backfill local sovereign truth.

## Exposure boundary

1. external exposure is controlled and minimal,
2. no automatic default publication path,
3. recoverability must remain possible without default external dependence.
