## 1) Style intent

Premium private command board с глубиной, тишиной и контролируемой космической атмосферой.
Стиль обязан раскрывать truth-surfaces, а не маскировать gaps.

## 2) Base rules (`ACTIVE`)

1. dark base surfaces;
2. violet metallic accents;
3. soft nebula glow at hierarchy edges;
4. restrained fog/depth layering;
5. deep metallic substrate, not flat black;
6. inner glow only where it communicates state.

## 3) Depth / fog / light logic (`RECOMMENDED`)

1. depth gradient: deepest at shell, lighter at active cards;
2. fog: only for background separation, never over text;
3. light anchors: primary action zones and critical alert points;
4. flow routes and owner-decision zones may receive controlled luminous emphasis.

## 4) Card surfaces

1. metallic dark glass cards with low-noise texture;
2. subtle border highlights for state changes;
3. status glows only for warning/critical/improving transitions;
4. morphology cards may be organ-like, but must preserve strict readability.

## 5) Edge treatment

1. rounded-corner command cards (medium radius);
2. restrained luminous edge on hover/focus;
3. no sharp neon outlines.

## 6) Typography mood

1. precise, technical, calm;
2. high readability for dense operational boards;
3. no playful display fonts.

## 7) Motion mood

1. smooth, low-frequency transitions;
2. pulse only for live risk/vital state;
3. avoid aggressive jitter or flashy bursts.

## 8) Acceptable cosmic references

1. nebula haze;
2. stellar depth gradients;
3. subtle orbital/ring motifs in vital signs;
4. cognition pathways and industrial signal vessels.

## 9) Forbidden excess

1. over-neon cyberpunk noise;
2. medieval ornamental chaos;
3. decorative animations without data meaning;
4. fake realtime visuals without source-backed change.
